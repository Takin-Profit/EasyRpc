# EasyRpc

(intetionally)  unsupported avro idl language syntax.

* @namespace
* @aliase
* fixed length fields
* @java-class
* @order
* null union shorthand, E.G.  `string?`
* throws,  use `union { SomeType, SomeError } ` instead
* oneway
* @logicalType
* default values , this should be defined in your code, not the api.
